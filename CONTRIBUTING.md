# Fluxo de Desenvolvimento

Para cada funcionalidade:

1. Planejar
2. Implementar
3. Executar a aplicação
4. Executar os testes
5. Revisar o código
6. Atualizar documentação
7. Commit
8. Push

## Padrão de Commit

Utilizar mensagens curtas e objetivas.

Exemplos:

Implementa impressão do tabuleiro

Adiciona teste de criação do tabuleiro

Refatora estrutura do projeto
