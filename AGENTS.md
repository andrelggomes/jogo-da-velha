# AGENTS.md

## Objetivo

Este projeto é um laboratório para aprender:

- Python
- Git
- GitHub
- GitHub Copilot
- Testes Automatizados
- Engenharia de Software

O jogo da velha é apenas o caso de estudo.

---

## Como trabalhar neste projeto

Antes de qualquer implementação:

1. Analise o código existente.
2. Explique a solução.
3. Justifique as decisões.
4. Faça pequenas alterações.
5. Preserve o comportamento atual.

---

## Estilo de código

- Python 3.14
- Código simples
- Funções pequenas
- Nomes claros
- Evitar complexidade desnecessária

---

## Sempre sugerir

- testes
- melhorias
- refatorações
- documentação

---

## Nunca

- implementar funcionalidades não solicitadas
- alterar arquitetura sem explicar
- adicionar dependências sem justificativa
